# Fine Wines Sales Intelligence Dashboard

## Files
- index.html
- styles.css
- js/app.js

Upload all three to the root of your GitHub Pages repository.

## Excel files used by the dashboard
- Group sales overall 2024.xlsx
- Group sales overall 2025.xlsx
- Shop top Hamper sales.xlsx (optional mapping file)

## Columns used from the group sales files
A SKU
B Description
C Pack Size
D Selling Price
E Sales Volume / Qty
G Case Volume
I Sales Value
L Profit
M GP%
Z Product Group
AA Product Type
AB Country
AE Department

## Hamper mapping file
Column A = 2025 SKU
Column B = 2024 SKU

Multiple SKUs can be mapped together. The dashboard combines totals before comparing 2024 vs 2025.
